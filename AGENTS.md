# 瀑布流管理平台 (Waterfall Management Platform)

## 项目概述

美柚广告系统 - 效果广告瀑布流管理模块，参考穿山甲/ToBid 平台设计，实现广告分组管理、广告源配置、数据分析和代码位ID管理功能。

## 系统结构

- **系统名称**：美柚广告系统
- **一级菜单**：流量管理
- **二级菜单**：流量分组管理、代码位ID管理

## 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4

## 核心功能

### 1. 页面切换
- 左侧导航支持：流量分组管理、代码位ID管理
- 当前页面高亮显示（粉色背景+右边框）

### 2. 瀑布流管理 - 广告场景管理
- 顶部导航支持多种广告场景：开屏、Banner、插屏、激励视频、原生
- 日期区间选择器
- 场景切换保留当前状态

### 3. 瀑布流管理 - 分组管理
- 添加分组（支持自定义名称和平台）
- 分组标签页快速切换
- 分组管理弹窗：拖拽排序调整优先级（使用 @dnd-kit/sortable）
- 分组管理弹窗：批量启用/关闭分组
- 分组开关控制
- 分组流量底价配置

### 4. 瀑布流管理 - 分组配置
- 基础信息展示（平台、广告位）
- 分组规则/受众定向（标签展示）
- 分组启用/禁用状态

### 5. 瀑布流管理 - 广告源管理
- 添加PID（名称、定价方式、价格）
- 批量操作：批量启用/停用DSP来源、批量设置价格
- 单个广告源状态开关
- 更多操作菜单（查看详情、编辑、复制、删除）
- 已启用/未启用分区折叠展示

### 6. 瀑布流管理 - 广告源表格字段
| 列名 | 说明 |
|------|------|
| □ | 复选框，支持全选 |
| 广告源名称 | 颜色圆点标识 + 名称 + 兜底标签 |
| 状态 | Switch开关（蓝色激活态） |
| 定价方式 | 三点菜单 + 竞价(蓝色标签)/定价(绿色标签) |
| 价格 | ¥xx.xx + 蓝色趋势图标（竞价模式显示-） |
| 预估收入(元) | 金额格式化 |
| eCPM(元) | 千次展示收益，带❓提示 |
| 千次请求价值 | 金额展示，带❓提示 |
| 请求量 | 数值格式化（支持万级显示） |
| 返回量 | 数值格式化 |
| 返回率 | 百分比展示，带❓提示 |
| 竞价成功数 | 数值格式化 |
| 竞胜率 | 百分比展示，带❓提示 |

### 7. 瀑布流管理 - 悬停详情
- 鼠标悬停广告源行显示详情卡片
- 展示核心配置和统计数据

### 8. 代码位ID管理 - 代码位列表
- 表格字段：代码位Id、系统平台（安卓蓝色/iOS紫色）、广告场景、广告位、绑定分组信息、操作
- 操作列：启用状态显示"× 关闭广告代码位Id"红色按钮，已关闭显示灰色"已关闭"
- 分页控件：首页/上一页/页码/下一页/末页、每页条数选择器（10/20/50）

### 9. 代码位ID管理 - 新增代码位弹窗
- 代码位Id（必填）
- 系统平台（安卓/iOS单选）
- 广告场景下拉选择
- 广告位Id输入
- 绑定分组下拉选择

### 10. A/B测试
- 两步弹窗流程：基本信息 → 配置测试组瀑布流
- 支持流量比例分配（A组对照组/B组测试组）
- 配置复制选项

### 11. 分组管理弹窗
- 点击"分组管理"按钮打开弹窗
- 拖拽排序调整分组优先级（使用 @dnd-kit/sortable）
- 默认分组固定在底部不可拖拽
- 支持全选非默认分组
- 批量启用/关闭分组

### 12. 批量操作DSP来源
- 选中DSP来源后点击"批量操作"按钮
- 操作类型：批量启用、批量停用、批量设置价格
- 批量设置价格时需输入价格值
- 操作确认后立即生效

```bash
pnpm build

# 生产启动
pnpm start
```

## 代码规范

- 遵循 TypeScript strict 模式
- 组件使用 'use client' 指令标记客户端组件
- 避免 Hydration 问题：不直接在 JSX 中使用 Date.now()、Math.random()
- 颜色值遵循 admin-design 规范：主色 #FF4D88，背景 #F2F2F5
- 拖拽排序使用 @dnd-kit/core + @dnd-kit/sortable
- JSX 中引号使用 `&ldquo;` / `&rdquo;` 转义（避免 react/no-unescaped-entities）

## 注意事项

- 服务端口：5000
- 开发环境自动热更新
- 使用 pnpm 作为包管理器（禁止使用 npm/yarn）
