# 美柚广告系统 - 瀑布流管理 PRD

## 1. 产品概述

### 1.1 产品名称
美柚广告系统 - 瀑布流管理模块

### 1.2 产品定位
参考穿山甲/ToBid 平台设计的效果广告瀑布流管理系统，实现广告分组管理、广告源配置、数据分析和代码位ID管理功能。

### 1.3 目标用户
- 广告运营人员
- 广告商务人员
- 数据分析师

### 1.4 系统结构
- **一级菜单**：流量管理
- **二级菜单**：瀑布流管理、代码位ID管理

## 3. 详细功能说明

### 3.1 瀑布流管理

#### 3.1.1 广告场景与广告位配置

| 广告场景 | 广告位ID | 广告位名称 |
|----------|----------|------------|
| 开屏 | 1000 | 美柚--开屏 |
| 插屏 | 2101 | 美柚-首页-插屏 |
| 插屏 | 2514 | 爱爱记录-记录完成插屏 |
| 信息流 | 1120 | 首页大社区feeds流 |
| 信息流 | 1601 | 美柚-她她圈-帖子详情楼间广告 |
| 信息流 | 1602 | 美柚-她她圈-帖子详情信息流 |

#### 3.1.2 子位配置（仅信息流场景）

| 广告位 | 子位选项 |
|--------|----------|
| 首页大社区feeds流 | 1、2、3 |
| 美柚-她她圈-帖子详情楼间广告 | 1、2、3、4 |
| 美柚-她她圈-帖子详情信息流 | 1、2 |

#### 3.1.3 分组管理

**功能要点：**
- 添加分组：输入分组名称、优先级
- 优先级规则：数值越小优先级越高，按升序排列展示
- 广告场景/广告位/子位：只读展示，自动带入顶部选择值
- 分组规则：支持多选配置
- 支持开启A/B测试

**分组展示格式：** `{优先级}-{分组名称}`

**A/B测试：**
- 开启后显示AB测试图标
- 支持配置测试组瀑布流
- 支持流量比例分配
- 保存草稿后显示"编辑A/B测试"按钮

#### 3.1.4 广告源管理

**广告源表格字段：**

| 字段名 | 说明 | 显示汇总 |
|--------|------|----------|
| 操作 | 查看详情、编辑、复制、删除 | 否 |
| 广告源名称 | 颜色圆点标识 + 名称 | 否 |
| 状态 | Switch开关（蓝色激活态） | 否 |
| 定价方式 | 竞价(蓝色)/定价(绿色)标签 | 否 |
| 价格 | ¥xx.xx | 否 |
| 预估收入(元) | 金额格式化 | 是 |
| eCPM(元) | 千次展示收益 | 是 |
| 千次请求价值 | 金额展示 | 是 |
| 请求量 | 数值格式化 | 是 |
| 返回量 | 数值格式化 | 是 |
| 返回率 | 百分比展示 | 是 |
| 竞价成功数 | 数值格式化 | 是 |
| 竞胜率 | 百分比展示 | 是 |

**汇总行规则：**
- 广告源名称列：显示 `{x}个广告源已启用`
- 状态、定价方式、价格列：显示为空
- 其他数值字段：显示汇总数据

**汇总行格式要求：**
- 千人均收益：`¥{value?.toFixed(2) || '-'}`
- 预估收入：保留1位小数，千分位格式化
- 返回率、竞价成功率：保留1位小数加%
- 值为0的字段：显示 `-`

**广告源分区：**
- 已启用广告源：正常展示
- 未启用广告源：可折叠展示

#### 3.1.5 添加/编辑广告源

**表单字段：**
- 广告代码位ID：下拉选择，格式 `{代码位ID}-{代码位名称}`
- 广告源名称：文本输入
- 定价方式：竞价/定价单选
- 价格：数值输入
- DSP来源：下拉选择

### 3.2 代码位ID管理

#### 3.2.1 代码位列表

**表格字段：**

| 字段名 | 说明 |
|--------|------|
| 代码位ID | 代码位唯一标识 |
| 名称 | 代码位名称 |
| 状态 | 开启(绿色)/停用(红色)标签 |
| 系统平台 | 安卓(蓝色)/iOS(紫色) |
| 广告场景 | 开屏/插屏/信息流 |
| 广告位 | 对应广告位名称 |
| 绑定分组信息 | 关联的分组名称 |
| 操作 | 编辑/启用/停用 |

**分页配置：**
- 每页条数：10/20/50 可选
- 首页/上一页/页码/下一页/末页

#### 3.2.2 新增/编辑代码位弹窗

**表单字段（按顺序）：**
1. 代码位ID（必填）
2. 名称（必填）
3. 系统平台：安卓/iOS单选
4. DSP来源：下拉选择
5. 广告场景：开屏/插屏/信息流
6. 广告位：根据场景动态显示
7. 价格模式：下拉选择

**广告位选项：**
- 开屏：美柚--开屏
- 插屏：美柚-首页-插屏、爱爱记录-记录完成插屏
- 信息流：首页大社区feeds流、美柚-她她圈-帖子详情楼间广告、美柚-她她圈-帖子详情信息流

## 5. 数据模型

### 5.1 广告分组 (AdGroup)

```typescript
interface AdGroup {
  id: string;
  name: string;                    // 分组名称
  priority: number;                // 优先级（数值越小越高）
  scene: string;                   // 广告场景
  slot: string;                    // 广告位
  slotName: string;                // 广告位名称
  subSlots?: number[];             // 子位列表
  enabled: boolean;                // 启用/禁用状态
  abTest?: {
    enabled: boolean;              // 是否开启A/B测试
    trafficSplit: number;          // 流量分配比例
    config?: object;              // A/B测试配置
  };
  rules?: GroupRule[];            // 分组规则
  floorPrice?: number;            // 流量底价
  sources?: AdSource[];           // 广告源列表
}
```

### 5.2 广告源 (AdSource)

```typescript
interface AdSource {
  id: string;
  name: string;                   // 广告源名称
  color: string;                   // 颜色标识
  status: boolean;                 // 启用/禁用状态
  pricingType: 'bidding' | 'fixed'; // 定价方式
  price: number;                  // 价格
  codePositionId: string;         // 代码位ID
  codePositionName: string;       // 代码位名称
  dspSource?: string;             // DSP来源
  // 统计数据
  estimatedRevenue?: number;      // 预估收入
  ecpm?: number;                 // eCPM
  requestValue?: number;          // 千次请求价值
  requests?: number;              // 请求量
  responses?: number;             // 返回量
  returnRate?: number;           // 返回率
  bidSuccessCount?: number;       // 竞价成功数
  bidWinRate?: number;            // 竞胜率
  impressions?: number;           // 展示量
  bidWinImpressionRate?: number; // 竞胜展示率
  clicks?: number;                // 点击数
  clickRate?: number;             // 点击率
  cpc?: number;                   // CPC
}
```

### 5.3 代码位 (CodePosition)

```typescript
interface CodePosition {
  id: string;
  codeId: string;                 // 代码位ID
  name: string;                   // 代码位名称
  status: 'active' | 'inactive'; // 状态
  platform: 'android' | 'ios';    // 系统平台
  scene: string;                  // 广告场景
  slot: string;                   // 广告位
  slotName: string;               // 广告位名称
  dspSource?: string;             // DSP来源
  priceMode?: string;             // 价格模式
  boundGroups?: string[];         // 绑定分组ID列表
}
```

## 7. 技术实现

### 7.1 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4

### 7.2 项目结构

```
├── src/
│   ├── app/
```
